## Start
- Open file **scripts/start.sh** on linux or **scripts/start_win.bat** on windows.
- Change **greenify_wp_docker_db_1** to **THE CONTAINER OF DATABASE**.
- Run file:
>+ Linux: bash scripts/start.sh
>+Windows: scripts/start_win.bat
